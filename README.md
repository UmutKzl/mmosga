# mmosga

Make macOS great again!

## Installation

First of all, if you don't have Homebrew installed, install that.

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Then, install some packages. We need them.

```bash
brew install gum git wget
```

Then, download and execute the script.

```bash
wget https://raw.githubusercontent.com/UmutKzl/mmosga/refs/heads/main/mmosga.sh
chmod +x mmosga.sh
./mmosga.sh
```
