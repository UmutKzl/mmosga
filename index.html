mmosga

A script to make default macOS much better.

Installation

Requirements: homebrew

NOTE: gum is needed too. But this script installs it automatically via homebrew. 

bash <(curl -s https://raw.githubusercontent.com/UmutKzl/mmosga/refs/heads/main/mmosga.sh)

Features

  • macOS Customizations by defaults command
  • Fancy confirmations and selections using gum
  • App installation
  • Rosetta 2 installation
  • Zsh, Ghostty, Neovim and Emacs configuration by a wizard
  • Ready system for developers and normal users
